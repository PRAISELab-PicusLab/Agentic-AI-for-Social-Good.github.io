# Agentic AI for Social Good

Special Track website @ ACM GoodIT 2026 — ACM SIGCAS International Conference on Information Technology for Social Good.

## Pages

- **Home** (`index.html`) — Scope, topics of interest, important dates
- **Submission** (`submission.html`) — Submission portal, guidelines, review process, ACM template
- **Organizers** (`organizers.html`) — Track organizers bios, Technical Program Committee

## Hosting

This site is designed to be hosted via [GitHub Pages](https://pages.github.com/). Simply enable GitHub Pages in the repository settings pointing to the `main` branch root.
